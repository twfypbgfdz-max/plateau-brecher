# Kalorien- und Ernaehrungs-App

Eigenstaendige Mobile-First-App zum einfachen Erfassen von Kalorien, Protein und Gewicht.

Ziel ist ein ruhiger, schneller Tages-Tracker ohne ueberladenes Fitnessstudio-Tool. Der erste Prototyp konzentriert sich auf wenige Kernwerte: kcal, Protein, Gewicht, Tagesuebersicht, Wochenuebersicht, einfache Charts und lokale Speicherung.

Die App bleibt getrennt von Personal-App und Public-App. Es werden keine Trainingsdaten, Sync-Logiken oder Plateau-Brecher-Dateien uebernommen.
