# 02_PUBLIC_APP

Dieser Bereich enthaelt die Public-App und bleibt strikt von der Personal-App getrennt.

- `aktuell` enthaelt aktuelle Public-Arbeitskopien.
- `tests` enthaelt Public-Testversionen.
- Falls ein lokaler `stable`-Ordner existiert, enthaelt er freigegebene Public-Stable-Versionen.
- `00_AKTUELL/public_latest` ist die Referenz fuer die letzte freigegebene Public-Stable/Live-Version.
- Keine Personal-Sync-, PR- oder Coach-Logik in die Public-App uebernehmen.
- Public-Versionen nur gezielt releasen.
