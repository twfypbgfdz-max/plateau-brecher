# 01_PERSONAL_APP

Dieser Bereich enthaelt die Personal-App und bleibt strikt von der Public-App getrennt.

- `tests` enthaelt aktive Test- und Arbeitsversionen.
- `stable` enthaelt freigegebene Stable-Versionen.
- `aktuell` enthaelt aeltere oder aktuelle Arbeitskopien und sollte nur bewusst genutzt werden.
- `changelogs` enthaelt Versionsaenderungen.
- Stable-Dateien nicht direkt bearbeiten.
